Comment ça marche
Trois étapes, zéro intervention

Connexion Shopify

Dès qu'un panier est abandonné, Callify vérifie en temps réel que la commande n'a pas été passée — puis déclenche la séquence d'appels.

FAQ
Questions fréquentes
Callify fonctionne-t-il avec toutes les boutiques Shopify ?
Oui — Callify est conçu exclusivement pour Shopify et se connecte nativement via l'API Shopify. Aucune intégration tierce, aucun code à installer manuellement. Compatible avec toutes les offres Shopify (Basic, Shopify, Advanced, Plus).
Que se passe-t-il si le client ne décroche pas ?
Callify effectue jusqu'à 3 tentatives espacées intelligemment (ex. T1 après 30 min, T2 après 3h, T3 le lendemain). Sans réponse sur les 3 appels, le panier est marqué "non joignable" et tracé dans votre tableau de bord. Aucun harcèlement, aucun appel hors plages légales.
Comment est configuré le script et la voix ?
Chaque déploiement est personnalisé par notre équipe : ton, vocabulaire, voix (genre, accent, chaleur), script, gestion des objections et offres conditionnelles (livraison gratuite, remise, urgence…). Vous validez le script avant la mise en production. C'est ce qui nous différencie d'un outil générique.
Les appels sont-ils conformes au RGPD ?
Oui. Callify n'appelle que les clients qui ont un panier actif et dont les données sont déjà dans votre Shopify — données que vous avez légalement collectées. Les appels sont traités sur des serveurs EU, les enregistrements peuvent être désactivés, et nous respectons les plages horaires légales de démarchage téléphonique.
Combien de temps prend l'installation ?
Moins de 48h entre votre demande et les premiers appels actifs. Étapes : appel de brief (30 min) → rédaction et validation du script (24h) → connexion Shopify et tests (quelques heures) → go live. Tout est géré par notre équipe — vous n'avez rien à toucher techniquement.
